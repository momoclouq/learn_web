padding: generate space around an element's content, inside of any defined borders. Tips: use padding with containers, not the content

document flow: the model by which elements are rendered by default in the CSS specifications. In other words, , block-level elements are displayed on a new line and inline elements on the same line. Everything is stacked in an ordered way from top to bottom
