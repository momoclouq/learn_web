Three different ways: float, position, flexbox

Yesterday, I checked floats and today I practiced it (check link). Now come the postioning

Positions:

  • relative: follows the default document flow, but can be offset
  • static (default): follows the default document flow, but can not be offset
  • absolute: the element is removed from the document flow, any offsets placed on will be relative to its parent. If the parent is static -> check grandparent, recursively
  • fixed: like absolute (removed from the document flow), but offsets are always positioned relative to the active viewport (truely fixed)

anchor div to set content as absolute position: set div to relative position

To use an anchor div: set the div to overflow: hidden; and min-height: ...px; to make the div visible

Flexbox in general:

  1. Turn the element in to flex: display: flex; or display: inline-flex; (apply to the container!)
  2. Now the content is adjustable according to flex rules (apply to the container!) such as: flex-direction, justify-content, align-items
  3. Flex-direction: row/column; -> justify-content: set the alignment of the content with the same direction of flex-direction
  4. align-items: set the alignment of the content with the different direction (row = up/down, column = right/left)
