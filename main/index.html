Positions:

Flexbox in general:

  1. Turn the element in to flex: display: flex; or display: inline-flex; (apply to the container!)
  2. Now the content is adjustable according to flex rules (apply to the container!) such as: flex-direction, justify-content, align-items
  3. Flex-direction: row/column; -> justify-content: set the alignment of the content with the same direction of flex-direction
  4. align-items: set the alignment of the content with the different direction (row = up/down, column = right/left)

More things about flexbox:

  • justify-content and align-items both have: flex-end (move the contents to the end side), flex-start(opposite of flex-end);
  • flex: reverse-row; = the items in the div will be displayed from right to left (flex-end is now left, start is right)
  • order: the default order is 0 for all items (this is applied to the content), increase it for certain contents will move them to the end (higher is further to the end)
  • order can be negative (-1, -2, etc).
  • To align individual items: align-sefl: flex-end, flex-start, center;

Responsive unit: relative to the length of another property.

  • absolute unit: pixels, inches, centimeters, points, etc
  • responsive unit: % (according to the container of the content)
  • em: 1em is equal to the default font-size for the element
  • rem: 1rem the same with 1em but the size is relative to root (html)
  • vh: 1vh = 1/100 view height
  • vw: 1vw = 1/100 view width
  • vmin: 1vmin = the minimum of 1vh and 1vw
  • vmax: 1vmax = the maximum ___

The grid system: a way to organize the content on your page in terms of rows of columns.

Example of using grid system

In order to use the grid system:

  • Create a container, with a "container" or "container-fluid" class
  • To create a row, use the "row" class
  • Inside the rows, use the column classes
  • col-12: this col will take up all spaces for 12 columns
  • col-md-8: this col will take up 8 columns when the screen is medium size or larger
  • the second div position will depend on the sum of the cols: is the first + second > 12, then the second div will go nextline

Size summary:

  • col- : All size
  • col-sm- :576px and more
  • col-md- :768px and more
  • col-lg- :992px and more
  • col-xl- :1200px and more

Example of columns auto-adjust to the available width

use "col" to auto adjust the remaining width
